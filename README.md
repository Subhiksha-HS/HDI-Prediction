# Human Development Index (HDI) Prediction

## 📌 Project Overview

The Human Development Index (HDI) Prediction project is a Machine Learning web application developed using Python and Flask. It predicts the Human Development Index category of a country based on four important socio-economic indicators:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- Gross National Income (GNI) per Capita

The project helps demonstrate how machine learning can be used to analyze development indicators and classify countries into different HDI categories.

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/Subhiksha-HS/HDI-Prediction.git
```

2. Navigate to the project folder

```
cd HDI-Prediction
```

3. Install the required libraries

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
